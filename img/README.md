# Card images

Drop your card PNGs here with these exact names and they'll appear automatically
in the duel (the app falls back to emoji art if a file is missing):

| File             | Card                          |
|------------------|-------------------------------|
| `marine.png`     | Marine, Birthday Guardian     |
| `alexis.png`     | Alexis the Mighty Baby        |
| `maman.png`      | Maman, Mad Chemistry Teacher  |
| `papa.png`       | Papa, Boardroom Tycoon        |

Tip: a tighter crop (just the card, less blue background) looks best, since the
image is scaled to fill the card slot. Square-ish or portrait crops work well.
